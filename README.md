# Password_Complexity_Checker

Primary purpose of this for me to practice programming in C.

Currently this application acts as a basic password complexity checker.

- The user inputs a password
- The password goes through multiple conditional checks.
- If the password is valid then the application exits succesfully
- Otherwise it will be called out
